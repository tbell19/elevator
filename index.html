Elevator Project

by Trent Bell

The Elevator I Chose

I chose the elevators closest to the enterance of the Texas Tech Library

The current design is a little busy and has a confusing button layout.

There are only four floors so there is no reason why the buttons cannot be stacked vertically.

Currently it is not intuitive to find the button for the floor that you are trying to go to with the buttons staggered left and right.

The elevator has lots of information but it is displayed in a way which can be overwhelming when trying to make a quick glance

There is very little seperation the buttons which are used often and the buttons which are only used in emergency situations

About this Elevator

This elevator is commonly used for

  • Taking students between floors
  • Transferring books and equipment between floors
  • Providing information about the library

Uncommon uses for this elevator

  • Emergency use by firefighters
  • Calling for help in the event of a malfunction
